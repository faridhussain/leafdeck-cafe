# Leaf Deck Café

A premium, editorial-inspired website for **Leaf Deck Café**, built with modern web technologies and designed to deliver a refined digital experience across all devices.

### Live Demo
https://leafdeck-cafe.vercel.app

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lenis
- Lucide React

## Features

- Elegant editorial-inspired UI
- Fully responsive design
- Smooth scrolling & animations
- Signature Dishes showcase
- Complete Menu & Bar Menu
- Catering page
- Contact page with form validation
- Google Maps integration
- Optimized images and fonts
- Accessible, mobile-first experience
